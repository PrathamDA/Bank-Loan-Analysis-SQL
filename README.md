🏦 Bank Loan Analysis SQL Project

About This Project

I created this project to strengthen my SQL skills by building a complete banking database from scratch. The project simulates how a bank stores and manages customer information, loan records, employee details, branch data, and payment history.

The main goal was not just to write SQL queries, but to understand how a real-world relational database is designed and used for data analysis.

---

Project Objectives

- Design a normalized relational database
- Build relationships using primary and foreign keys
- Insert realistic banking data
- Practice SQL queries from basic to advanced
- Implement Views, Stored Procedures, and Triggers
- Develop a project suitable for my SQL portfolio

---
